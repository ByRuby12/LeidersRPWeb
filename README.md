# 🚨 LEIDERSRP | SITIO WEB OFICIAL 🌐

## Bienvenido a la plataforma oficial de LeidersRP

LeidersRP es una comunidad de **Roleplay en consola (PS4 y PS5)** activa desde 2020, pionera en ofrecer una experiencia de rol estructurada, seria y adaptada completamente a jugadores de consola en GTA V.

Este sitio web representa el siguiente paso en la evolución del servidor.
